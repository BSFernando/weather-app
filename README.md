# Passo a passo

Clone o repositório
```bash
git clone https://github.com/BSFernando/weather-app
cd weather-app
```

Instale as dependências
```bash
npm install
```

Iniciar o servidor local

```bash
ng serve
```

Acesse
```
http://localhost:4200
```
